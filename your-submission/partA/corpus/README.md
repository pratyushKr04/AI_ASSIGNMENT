# Evaluation Corpus

**Source:** FLORES-200 devtest split (facebook/flores on HuggingFace)
**Sentences per language:** 1012
**Languages:** English (eng), Hindi (hin), Tamil (tam), Kannada (kan)
**Parallel:** Yes — line N in every file is the same sentence.

## Domain & Caveats

- FLORES-200 is drawn from English Wikipedia and Wikinews, then
  professionally translated into 200+ languages.
- Domain: general knowledge / news. Not conversational.
- Caveat: translationese may slightly inflate target-language
  complexity (translators may use more formal/verbose phrasing).
